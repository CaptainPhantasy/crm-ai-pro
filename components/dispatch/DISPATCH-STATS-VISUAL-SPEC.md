### Buttons

### Chart Interactions
```
Donut Chart:    - Hover: Tooltip shows exact count
                - Click: Future feature (filter map)

Line Chart:     - Hover: Tooltip shows time and active count
                - Crosshair: Shows vertical line at cursor

Bar Chart:      - Hover: Tooltip shows tech name and miles
                - Click: Future feature (show tech details)

Progress Bars:  - Hover: No interaction
                - Click: Future feature (show tech details)
```

## Accessibility Features

### Keyboard Navigation
```
Tab Order:
1. Time Range buttons (Today, Week, Month)
2. Export PDF button
3. Export CSV button
4. Refresh button
5. Collapse/Expand button
6. (When expanded): Chart interactive elements

Shortcuts:
- Space/Enter: Activate button
- Tab: Next element
- Shift+Tab: Previous element
- Escape: Close any open tooltips
```
