--------------------------------------------------------------------------------
PAGE: INBOX
--------------------------------------------------------------------------------
URL: /inbox
ACCESS: Owner, Admin
NAVIGATE COMMAND: navigate(page: "inbox")

DESCRIPTION:
The central communications hub. All customer messages, emails, and internal
notes flow through here.

KEY FEATURES:
- Unified inbox for all channels (email, form submissions)
- AI-suggested responses with one-click sending
- Conversation threading by contact
- Tags and filters for organization
- Search across all messages
- Mark as read/unread, star important items
- Link conversations to jobs

WHAT USERS CAN DO:
- Reply to customer messages
- Draft responses with AI assistance
- Create jobs from conversations
- Tag and categorize messages
- Search message history

PAGE: JOBS
--------------------------------------------------------------------------------
URL: /jobs
ACCESS: Owner, Admin, Dispatcher
NAVIGATE COMMAND: navigate(page: "jobs")

DESCRIPTION:
Work order management central. View, create, edit, and manage all jobs.

KEY FEATURES:
- List view of all jobs with sorting/filtering
- Status tabs: Lead, Scheduled, In Progress, Completed, Invoiced, Paid
- Create new job dialog
- Quick assign technician
- Bulk actions (assign, reschedule, delete)
- Job detail sidebar with full info
- Search by customer, address, description

WHAT USERS CAN DO:
- Create new jobs
- Assign technicians
- Change job status
- Reschedule appointments
- Add materials and notes
- Generate invoices
- View job history and photos
